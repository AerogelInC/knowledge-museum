# 网站部署指南

## 📋 快速部署方案

你的网站是纯静态网站（HTML + CSS + JavaScript），可以轻松部署到多个免费平台。

## 🌟 推荐方案（支持实时更新）

### 方案一：Netlify（最简单推荐 ⭐）

**优点：**
- ✅ 完全免费
- ✅ 拖拽即可部署，无需命令行
- ✅ 支持实时更新（修改文件后重新拖拽即可）
- ✅ 自动获得 HTTPS 证书
- ✅ 可以绑定自定义域名

**部署步骤：**

1. **访问 Netlify**
   - 打开 https://www.netlify.com
   - 点击 "Sign up" 注册账号（可以用 GitHub、邮箱等注册）

2. **部署网站**
   - 登录后，在首页找到 "Want to deploy a new site without connecting to Git?"
   - 直接将整个项目文件夹（包含 index.html、styles.css、script.js）拖拽到页面上的虚线框内
   - 等待几秒钟，网站就部署完成了！

3. **获取网站链接**
   - 部署完成后，会显示类似 `https://random-name-123456.netlify.app` 的链接
   - 这个链接就是你的网站地址，可以分享给任何人

4. **实时更新网站**
   - 修改本地文件后，再次拖拽整个文件夹到 Netlify
   - 或登录 Netlify 后台，点击 "Deploys" → "Drop files here" 上传新文件
   - 几分钟内更新就会生效

---

### 方案二：Vercel（同样简单）

**优点：**
- ✅ 完全免费
- ✅ 界面简洁美观
- ✅ 部署速度快
- ✅ 支持实时更新

**部署步骤：**

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 点击 "Sign Up" 注册账号

2. **部署网站**
   - 登录后，点击 "Add New..." → "Project"
   - 如果使用拖拽方式：直接拖拽项目文件夹
   - 或点击 "Browse" 选择项目文件夹
   - 点击 "Deploy"

3. **获取网站链接**
   - 部署完成后会显示网站链接，类似 `https://your-project.vercel.app`

4. **更新网站**
   - 修改文件后重新拖拽部署，或使用 Vercel CLI（可选）

---

### 方案三：GitHub Pages（适合程序员）

**优点：**
- ✅ 完全免费
- ✅ 与 GitHub 集成
- ✅ 通过 Git 推送更新

**部署步骤：**

1. **安装 Git**（如果还没安装）
   - 下载：https://git-scm.com/download/win

2. **创建 GitHub 仓库**
   - 访问 https://github.com
   - 注册/登录账号
   - 点击右上角 "+" → "New repository"
   - 输入仓库名（如 `knowledge-museum`）
   - 选择 Public
   - 点击 "Create repository"

3. **上传文件到 GitHub**
   - 在项目文件夹右键 → "Git Bash Here"（或打开 PowerShell）
   - 执行以下命令：
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/你的用户名/仓库名.git
   git push -u origin main
   ```

4. **启用 GitHub Pages**
   - 在 GitHub 仓库页面，点击 "Settings"
   - 左侧找到 "Pages"
   - Source 选择 "main" 分支
   - 点击 "Save"
   - 几分钟后，网站会在 `https://你的用户名.github.io/仓库名` 上线

5. **更新网站**
   - 修改文件后，执行：
   ```bash
   git add .
   git commit -m "Update website"
   git push
   ```
   - 几分钟后更新就会生效

## ❓ 常见问题

**Q: 部署后如何修改网站？**
A: 修改本地文件后，重新拖拽到 Netlify/Vercel，或使用 Git 推送更新。

**Q: 网站是永久免费的吗？**
A: 是的，以上平台的基础服务都是永久免费的。

**Q: 可以自定义域名吗？**
A: 可以！Netlify 和 Vercel 都支持绑定自己的域名（需要购买域名）。

**Q: 网站访问速度如何？**
A: 这些平台都有全球 CDN，访问速度很快。
